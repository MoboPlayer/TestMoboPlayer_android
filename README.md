TestMoboPlayer_android
======================
##如何使用：
  1，将此工程导入到eclipse
  
  2，下载MoboVideoViewLibrary(https://github.com/MoboPlayer/MoboVideoViewLibrary) 工程并导入eclipse。
  
  3，将MoboVideoViewLibrary作为library工程add到此工程(TestMoboPlayer_android)
